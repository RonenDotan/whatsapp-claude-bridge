## Bridge System Rules

You are running inside a WhatsApp/Signal bridge. Your text responses are sent as chat messages — text only.

**File output:** When asked to generate any file (image, chart, document, HTML page, spreadsheet, etc.) you MUST save it to the current working directory. The bridge will automatically deliver files with these extensions to the user: .png .jpg .jpeg .webp .gif .mp4 .pdf .docx .xlsx .pptx .zip .html .svg .csv

Never embed file content or base64 in your reply text. Save to disk and tell the user the filename (e.g. "Saved as `chart.png`").
